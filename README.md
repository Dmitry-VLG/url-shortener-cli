# URL Shortener CLI

Простой CLI-сервис сокращения ссылок. Реализовано:
* Генерация уникальных коротких кодов (`clck.ru/<code>`)
* Пер-пользовательская уникальность (UUID)
* Лимит переходов и TTL (авто-удаление «протухших» ссылок)
* Открытие ссылки в браузере
* Очистка устаревших ссылок в фоне
* Юнит- и интеграционные тесты (JUnit 5)
